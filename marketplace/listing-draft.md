# Ecomail for Magento 2

## Short Description

Connect Magento 2 with Ecomail for email marketing, customer synchronization, ecommerce tracking, abandoned cart campaigns, and order data.

## Long Description

Ecomail for Magento 2 helps merchants connect their Magento store with Ecomail email marketing and automation services. The extension synchronizes newsletter subscribers, customer data, order transactions, cart contents, and selected storefront tracking events into Ecomail.

The extension is configured directly from Magento admin. Store owners can choose the Ecomail subscriber list, select which customer fields should be shared, enable or disable tracking features, run an initial synchronization of existing customers and orders, and view recent Ecomail API request statuses.

## Key Features

- Newsletter subscriber sync.
- Configurable Ecomail contact source.
- Customer data sync with optional name, address, birthday, customer group, and language fields.
- Order transaction sync.
- Cart item sync for abandoned cart automation.
- Product view and page view tracking.
- Magento cookie consent support for browser tracking.
- Checkout newsletter opt-out checkbox.
- Token-protected inbound webhook for subscription status updates.
- Cron-backed initial sync with progress display.
- Recent API request log in Magento admin.
- Safe tag handling for existing Ecomail contacts.
