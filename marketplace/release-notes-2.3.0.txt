Ecomail for Magento 2 version 2.3.0

This release modernizes the Ecomail Magento 2 integration for current Magento 2.4.x installations.

Highlights:
- Composer package metadata prepared for Adobe Commerce Marketplace submission.
- Newsletter subscriber synchronization with Ecomail lists.
- Customer, order, cart, and product tracking synchronization.
- Cron-backed initial sync for existing customers and orders.
- Admin progress panel for initial sync.
- Recent Ecomail API request status panel in Magento admin.
- Token-protected inbound webhook for subscription status updates.
- Optional checkout newsletter opt-out checkbox.
- Cookie consent support for storefront tracking scripts.
- Safer Ecomail tag handling that preserves existing tags for single-contact updates.
- Customer group tags are normalized without spaces, and Magento NOT LOGGED IN is not sent as a tag.
- Floating unsaved changes warning in Ecomail configuration.

Notes:
- Magento cron must be running for initial sync.
- Magento 2.4 requires a working search service such as OpenSearch or Elasticsearch.
- Bulk initial sync can update existing Ecomail contacts when enabled in configuration.
